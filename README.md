# Unraid Docker Templates

Docker container templates for Unraid Community Applications.

## Available Templates

| Application | Description | Docker Hub |
|-------------|-------------|------------|
| [Prunerr](templates/prunerr.xml) | Media library cleanup tool for Plex, Sonarr, and Radarr | [helliott20/prunerr](https://hub.docker.com/r/helliott20/prunerr) |

## Installation

### Adding this repository to Unraid

1. Go to the **Docker** tab in Unraid
2. Scroll down to **Template Repositories**
3. Add: `https://github.com/helliott20/unraid-templates`
4. Click **Save**

### Manual Installation

You can also use individual template URLs directly:

```
https://raw.githubusercontent.com/helliott20/unraid-templates/main/templates/prunerr.xml
```
